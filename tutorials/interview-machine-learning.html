1. ML Fundamentals

Q1: What is Machine Learning?

Machine Learning (ML) is a subset of AI that enables systems to learn and improve from experience without being explicitly programmed. It uses algorithms to find patterns in data and make predictions.

Types of ML:

  • Supervised Learning: Learning with labeled data
  • Unsupervised Learning: Learning with unlabeled data
  • Reinforcement Learning: Learning through rewards/penalties
  • Semi-supervised Learning: Combination of labeled and unlabeled data
Q2: What is the difference between classification and regression?
Aspect Classification Regression
Output Discrete categories Continuous values
Goal Predict class labels Predict numerical values
Examples Spam detection, image classification Price prediction, temperature forecasting
Q3: What is the difference between parametric and non-parametric models?
Aspect Parametric Non-parametric
Parameters Fixed number of parameters Flexible number of parameters
Complexity Independent of data size Grows with data size
Examples Linear regression, logistic regression KNN, decision trees
Q4: What is the difference between generative and discriminative models?
Aspect Generative Discriminative
Goal Model joint distribution P(X,Y) Model conditional distribution P(Y|X)
Use Case Generate new samples Classify/predict
Examples Naive Bayes, GANs Logistic regression, SVM
Q5: What is the curse of dimensionality?

Curse of Dimensionality refers to various phenomena that arise when analyzing data in high-dimensional spaces. As dimensions increase, data becomes sparse, and distance metrics become less meaningful.

Effects:

  • Exponential increase in data needed
  • Distance concentration
  • Overfitting risk increases
  • Computational complexity increases

2. Supervised Learning

Q6: What is linear regression?

Linear Regression is a supervised learning algorithm used for predicting continuous values. It models the relationship between input features and output as a linear equation.

# Linear regression with scikit-learn
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
model = LinearRegression()
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Q7: What is logistic regression?

Logistic Regression is a classification algorithm that uses the sigmoid function to predict probabilities. It's used for binary classification problems.

# Logistic regression
from sklearn.linear_model import LogisticRegression

model = LogisticRegression()
model.fit(X_train, y_train)
predictions = model.predict(X_test)
probabilities = model.predict_proba(X_test)
Q8: What is the difference between linear and logistic regression?
Aspect Linear Regression Logistic Regression
Type Regression Classification
Output Continuous values Probabilities (0-1)
Function Linear equation Sigmoid function
Q9: What is a decision tree?

Decision Tree is a supervised learning algorithm that uses a tree-like model of decisions. It splits data based on features to make predictions.

# Decision tree classifier
from sklearn.tree import DecisionTreeClassifier

model = DecisionTreeClassifier(max_depth=3)
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Q10: What is a random forest?

Random Forest is an ensemble method that combines multiple decision trees. It reduces overfitting by averaging predictions from many trees.

# Random forest
from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Q11: What is gradient boosting?

Gradient Boosting is an ensemble technique that builds models sequentially, each correcting errors of the previous one. It's powerful for tabular data.

# Gradient boosting
from sklearn.ensemble import GradientBoostingClassifier

model = GradientBoostingClassifier(n_estimators=100, learning_rate=0.1)
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Q12: What is XGBoost?

XGBoost is an optimized gradient boosting library known for its performance and speed. It includes regularization and handles missing values.

# XGBoost
import xgboost as xgb

model = xgb.XGBClassifier(n_estimators=100, learning_rate=0.1)
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Q13: What is SVM?

Support Vector Machine (SVM) is a supervised learning algorithm that finds the optimal hyperplane to separate classes. It works well for high-dimensional data.

# SVM classifier
from sklearn.svm import SVC

model = SVC(kernel='rbf', C=1.0)
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Q14: What is KNN?

K-Nearest Neighbors (KNN) is a simple algorithm that classifies data points based on the majority class of their k nearest neighbors.

# KNN classifier
from sklearn.neighbors import KNeighborsClassifier

model = KNeighborsClassifier(n_neighbors=5)
model.fit(X_train, y_train)
predictions = model.predict(X_test)
Q15: What is Naive Bayes?

Naive Bayes is a probabilistic classifier based on Bayes' theorem with the assumption of feature independence. It's fast and works well with text data.

# Naive Bayes
from sklearn.naive_bayes import GaussianNB

model = GaussianNB()
model.fit(X_train, y_train)
predictions = model.predict(X_test)

3. Unsupervised Learning

Q16: What is clustering?

Clustering is the task of grouping similar data points together. It's an unsupervised learning technique used for exploratory data analysis.

Q17: What is K-means clustering?

K-means is an iterative algorithm that partitions data into K clusters by minimizing the within-cluster sum of squares.

# K-means clustering
from sklearn.cluster import KMeans

kmeans = KMeans(n_clusters=3, random_state=42)
kmeans.fit(X)
labels = kmeans.labels_
centroids = kmeans.cluster_centers_
Q18: What is hierarchical clustering?

Hierarchical Clustering builds a hierarchy of clusters either in a bottom-up (agglomerative) or top-down (divisive) manner.

# Hierarchical clustering
from sklearn.cluster import AgglomerativeClustering

model = AgglomerativeClustering(n_clusters=3)
labels = model.fit_predict(X)
Q19: What is DBSCAN?

DBSCAN is a density-based clustering algorithm that groups points based on density. It can find clusters of arbitrary shapes and identify outliers.

# DBSCAN clustering
from sklearn.cluster import DBSCAN

dbscan = DBSCAN(eps=0.5, min_samples=5)
labels = dbscan.fit_predict(X)
Q20: What is PCA?

Principal Component Analysis (PCA) is a dimensionality reduction technique that transforms data into orthogonal components that capture maximum variance.

# PCA
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
X_pca = pca.fit_transform(X)
print(f"Explained variance: {pca.explained_variance_ratio_}")
Q21: What is t-SNE?

t-SNE is a nonlinear dimensionality reduction technique used for visualizing high-dimensional data in 2D or 3D space.

# t-SNE
from sklearn.manifold import TSNE

tsne = TSNE(n_components=2, random_state=42)
X_tsne = tsne.fit_transform(X)
Q22: What is the difference between PCA and t-SNE?
Aspect PCA t-SNE
Type Linear Nonlinear
Purpose Dimensionality reduction Visualization
Preserves Global structure Local structure

4. Model Evaluation

Q23: What is accuracy?

Accuracy is the ratio of correct predictions to total predictions. It's not suitable for imbalanced datasets.

# Accuracy
from sklearn.metrics import accuracy_score

accuracy = accuracy_score(y_test, predictions)
print(f"Accuracy: {accuracy:.2f}")
Q24: What is precision and recall?

Precision is the ratio of true positives to total predicted positives. Recall is the ratio of true positives to total actual positives.

# Precision and recall
from sklearn.metrics import precision_score, recall_score

precision = precision_score(y_test, predictions)
recall = recall_score(y_test, predictions)
Q25: What is F1 score?

F1 Score is the harmonic mean of precision and recall. It provides a balanced measure for classification performance.

# F1 score
from sklearn.metrics import f1_score

f1 = f1_score(y_test, predictions)
Q26: What is ROC curve and AUC?

ROC Curve plots true positive rate vs false positive rate at various thresholds. AUC is the area under the ROC curve, measuring model performance.

# ROC AUC
from sklearn.metrics import roc_auc_score, roc_curve

auc = roc_auc_score(y_test, probabilities[:, 1])
fpr, tpr, thresholds = roc_curve(y_test, probabilities[:, 1])
Q27: What is confusion matrix?

Confusion Matrix is a table showing true positives, true negatives, false positives, and false negatives.

# Confusion matrix
from sklearn.metrics import confusion_matrix

cm = confusion_matrix(y_test, predictions)
print(cm)
Q28: What is mean squared error?

Mean Squared Error (MSE) measures the average squared difference between predicted and actual values. It's used for regression evaluation.

# MSE
from sklearn.metrics import mean_squared_error

mse = mean_squared_error(y_test, predictions)
Q29: What is R-squared?

R-squared measures the proportion of variance in the dependent variable explained by the model. Values range from 0 to 1.

# R-squared
from sklearn.metrics import r2_score

r2 = r2_score(y_test, predictions)

5. Feature Engineering

Q30: What is one-hot encoding?

One-Hot Encoding converts categorical variables into binary vectors, where each category becomes a separate column.

# One-hot encoding
from sklearn.preprocessing import OneHotEncoder

encoder = OneHotEncoder(sparse=False)
encoded = encoder.fit_transform(df[['category']])
Q31: What is label encoding?

Label Encoding converts categorical variables into numerical labels. It's suitable for ordinal data but can introduce unintended ordinal relationships.

# Label encoding
from sklearn.preprocessing import LabelEncoder

encoder = LabelEncoder()
encoded = encoder.fit_transform(df['category'])
Q32: What is standardization?

Standardization scales features to have zero mean and unit variance. It's important for algorithms sensitive to feature scales.

# Standardization
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
Q33: What is normalization?

Normalization scales features to a range, typically [0, 1]. It's useful when features have different scales.

# Normalization
from sklearn.preprocessing import MinMaxScaler

scaler = MinMaxScaler()
X_normalized = scaler.fit_transform(X)
Q34: What is the difference between standardization and normalization?
Aspect Standardization Normalization
Range No fixed range (mean=0, std=1) Fixed range (usually [0,1])
Effect Preserves outliers Sensitive to outliers
Use Case Normal distribution Bounded features

6. Model Optimization

Q35: What is hyperparameter tuning?

Hyperparameter Tuning is the process of finding the optimal hyperparameters for a model to improve performance.

# Grid search
from sklearn.model_selection import GridSearchCV

param_grid = {'n_estimators': [50, 100, 200], 'max_depth': [3, 5, 7]}
grid_search = GridSearchCV(RandomForestClassifier(), param_grid, cv=5)
grid_search.fit(X_train, y_train)
Q36: What is the difference between grid search and random search?
Aspect Grid Search Random Search
Search Exhaustive Random sampling
Efficiency Less efficient More efficient
Best For Small parameter space Large parameter space
Q37: What is Bayesian optimization?

Bayesian Optimization is a sequential design strategy for global optimization of black-box functions. It builds a probabilistic model of the function and uses it to select the next hyperparameters to evaluate.

Q38: What is learning rate?

Learning Rate controls how much the model changes in response to the estimated error. Too high can cause overshooting, too low can cause slow convergence.

Q39: What is the difference between batch, stochastic, and mini-batch gradient descent?
Type Data Used Updates
Batch GD Entire dataset One per epoch
Stochastic GD One sample One per sample
Mini-batch GD Batch of samples One per batch

7. ML Algorithms

Q40: What is the difference between bagging and boosting?

Q41: What is AdaBoost?

AdaBoost is an ensemble method that combines weak learners into a strong learner by focusing on difficult examples. It adjusts weights based on prediction errors.

# AdaBoost
from sklearn.ensemble import AdaBoostClassifier

model = AdaBoostClassifier(n_estimators=50)
model.fit(X_train, y_train)
Q42: What is LightGBM?

LightGBM is a gradient boosting framework that uses tree-based learning algorithms. It's known for its speed and efficiency with large datasets.

# LightGBM
import lightgbm as lgb

model = lgb.LGBMClassifier(n_estimators=100)
model.fit(X_train, y_train)
Q43: What is CatBoost?

CatBoost is a gradient boosting algorithm that handles categorical features natively without extensive preprocessing. It's robust and provides good performance.

# CatBoost
from catboost import CatBoostClassifier

model = CatBoostClassifier(iterations=100)
model.fit(X_train, y_train, cat_features=categorical_features)

8. ML in Practice

Q45: What is the ML pipeline?

ML Pipeline is a sequence of data processing steps including data collection, preprocessing, feature engineering, model training, evaluation, and deployment.

# ML pipeline with scikit-learn
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.ensemble import RandomForestClassifier

pipeline = Pipeline([
    ('scaler', StandardScaler()),
    ('classifier', RandomForestClassifier())
])

pipeline.fit(X_train, y_train)
Q46: What is model deployment?

Model Deployment is the process of making a trained model available for use in production. It involves serialization, API creation, and monitoring.

# Save model with joblib
import joblib

joblib.dump(model, 'model.pkl')

# Load model
loaded_model = joblib.load('model.pkl')
Q47: What is model monitoring?

Model Monitoring involves tracking model performance in production to detect degradation, data drift, and concept drift. It ensures the model continues to perform well.

Q48: What is data drift?

Data Drift occurs when the statistical properties of input data change over time, causing model performance to degrade. It requires retraining or updating the model.

Q49: What is A/B testing for ML models?

A/B Testing compares two versions of a model to determine which performs better. It randomly assigns users to different model versions and measures key metrics.
