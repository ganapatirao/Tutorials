Chapter 1: Introduction to C#

1.1 What is C#?

C# (pronounced "C-sharp") is a modern, object-oriented programming language developed by Microsoft as part of the .NET initiative. It was designed for building a wide range of applications that run on the .NET platform.

1.2 History of C#

C# was developed by Microsoft and led by Anders Hejlsberg. It was first released in 2000 as part of Microsoft's .NET Framework. Since then, it has evolved through multiple versions, with C# 12 being the latest major release.

1.3 Key Features

  • Object-Oriented: Supports encapsulation, inheritance, and polymorphism
  • Type-Safe: Strongly typed language with compile-time type checking
  • Component-Oriented: Supports properties, methods, events, and attributes
  • Garbage Collection: Automatic memory management
  • Modern Language Features: LINQ, async/await, pattern matching, records
  • Cross-Platform: Runs on Windows, Linux, and macOS via .NET Core/.NET 5+

1.4 C# Applications

  • Desktop applications (Windows Forms, WPF, MAUI)
  • Web applications (ASP.NET, Blazor)
  • Mobile applications (Xamarin, MAUI)
  • Game development (Unity)
  • Cloud services (Azure Functions)
  • Microservices

Chapter 3: Data Types and Variables

3.1 Value Types

Value types store the actual data directly on the stack:

// Integral types
byte b = 255;           // 8-bit unsigned (0 to 255)
sbyte sb = -128;        // 8-bit signed (-128 to 127)
short s = 32767;        // 16-bit signed
ushort us = 65535;      // 16-bit unsigned
int i = 2147483647;     // 32-bit signed
uint ui = 4294967295U;  // 32-bit unsigned
long l = 9223372036854775807L;   // 64-bit signed
ulong ul = 18446744073709551615UL; // 64-bit unsigned

// Floating-point types
float f = 3.14f;        // 32-bit (7 digits precision)
double d = 3.14159265359; // 64-bit (15-16 digits precision)

// Other types
bool flag = true;       // Boolean
char c = 'A';           // 16-bit Unicode character
decimal money = 100.50m; // 128-bit (28-29 digits precision)

3.2 Reference Types

Reference types store a reference to the data on the heap:

// String
string name = "John Doe";

// Class
Person person = new Person();

// Array
int[] numbers = new int[5];

// Interface
IEnumerable collection = new List();

3.3 Variable Declaration and Initialization

// Explicit typing
int age = 25;
string name = "John";

// Type inference (var)
var count = 10;         // Compiler infers int
var message = "Hello";  // Compiler infers string

// Multiple declaration
int x = 10, y = 20, z = 30;

// Default values
int a = default;         // 0
string s = default;     // null
bool b = default;       // false

3.4 Constants

// const - compile-time constant
const double Pi = 3.14159;
const int MaxUsers = 100;

// readonly - runtime constant (can be set in constructor)
class Configuration
{
    public readonly string ApiKey;
    
    public Configuration(string key)
    {
        ApiKey = key;
    }
}

Chapter 4: Control Flow

4.1 Conditional Statements

// if-else
int score = 85;

if (score >= 90)
{
    Console.WriteLine("Grade: A");
}
else if (score >= 80)
{
    Console.WriteLine("Grade: B");
}
else if (score >= 70)
{
    Console.WriteLine("Grade: C");
}
else
{
    Console.WriteLine("Grade: F");
}

// switch statement
string day = "Monday";

switch (day)
{
    case "Monday":
        Console.WriteLine("Start of the week");
        break;
    case "Friday":
        Console.WriteLine("End of the week");
        break;
    default:
        Console.WriteLine("Mid week");
        break;
}

// switch expression (C# 8+)
string grade = score switch
{
    >= 90 => "A",
    >= 80 => "B",
    >= 70 => "C",
    _ => "F"
};

4.2 Loops

// for loop
for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
}

// while loop
int count = 0;
while (count < 10)
{
    Console.WriteLine(count);
    count++;
}

// do-while loop
int num = 0;
do
{
    Console.WriteLine(num);
    num++;
} while (num < 10);

// foreach loop
int[] numbers = { 1, 2, 3, 4, 5 };
foreach (int number in numbers)
{
    Console.WriteLine(number);
}

4.3 Jump Statements

// break - exit loop
for (int i = 0; i < 10; i++)
{
    if (i == 5)
        break;
    Console.WriteLine(i);
}

// continue - skip to next iteration
for (int i = 0; i < 10; i++)
{
    if (i % 2 == 0)
        continue;
    Console.WriteLine(i);
}

// return - exit method
int Calculate(int x, int y)
{
    return x + y;
}

Chapter 5: Object-Oriented Programming

5.1 Classes and Objects

public class Person
{
    // Fields
    private string _name;
    private int _age;
    
    // Constructor
    public Person(string name, int age)
    {
        _name = name;
        _age = age;
    }
    
    // Properties
    public string Name
    {
        get { return _name; }
        set { _name = value; }
    }
    
    public int Age
    {
        get { return _age; }
        set { _age = value; }
    }
    
    // Method
    public void Introduce()
    {
        Console.WriteLine($"Hi, I'm {_name} and I'm {_age} years old.");
    }
}

// Creating objects
Person person = new Person("John", 30);
person.Introduce();

5.2 Inheritance

// Base class
public class Animal
{
    public string Name { get; set; }
    
    public Animal(string name)
    {
        Name = name;
    }
    
    public virtual void MakeSound()
    {
        Console.WriteLine("Animal makes a sound");
    }
}

// Derived class
public class Dog : Animal
{
    public string Breed { get; set; }
    
    public Dog(string name, string breed) : base(name)
    {
        Breed = breed;
    }
    
    public override void MakeSound()
    {
        Console.WriteLine("Woof!");
    }
    
    public void Fetch()
    {
        Console.WriteLine($"{Name} is fetching the ball!");
    }
}

5.3 Polymorphism

// Method overloading (compile-time polymorphism)
public class Calculator
{
    public int Add(int a, int b)
    {
        return a + b;
    }
    
    public double Add(double a, double b)
    {
        return a + b;
    }
    
    public int Add(int a, int b, int c)
    {
        return a + b + c;
    }
}

// Method overriding (runtime polymorphism)
Animal animal = new Dog("Buddy", "Golden Retriever");
animal.MakeSound(); // Outputs: Woof!

5.4 Abstraction

// Abstract class
public abstract class Shape
{
    public abstract double GetArea();
    public abstract double GetPerimeter();
    
    public void DisplayInfo()
    {
        Console.WriteLine($"Area: {GetArea()}");
        Console.WriteLine($"Perimeter: {GetPerimeter()}");
    }
}

// Concrete class
public class Circle : Shape
{
    public double Radius { get; set; }
    
    public Circle(double radius)
    {
        Radius = radius;
    }
    
    public override double GetArea()
    {
        return Math.PI * Radius * Radius;
    }
    
    public override double GetPerimeter()
    {
        return 2 * Math.PI * Radius;
    }
}

// Interface
public interface IResizable
{
    void Resize(double factor);
}

public class Rectangle : Shape, IResizable
{
    public double Width { get; set; }
    public double Height { get; set; }
    
    public Rectangle(double width, double height)
    {
        Width = width;
        Height = height;
    }
    
    public override double GetArea() => Width * Height;
    public override double GetPerimeter() => 2 * (Width + Height);
    
    public void Resize(double factor)
    {
        Width *= factor;
        Height *= factor;
    }
}

{

    {
        get { return _items[index]; }

    }

6.3 Extension Methods

public static class StringExtensions
{
    public static bool IsNullOrEmpty(this string value)
    {
        return string.IsNullOrEmpty(value);
    }
    
    public static string ToTitleCase(this string value)
    {
        return CultureInfo.CurrentCulture.TextInfo
            .ToTitleCase(value.ToLower());
    }
    
    public static string Reverse(this string value)
    {
        char[] charArray = value.ToCharArray();
        Array.Reverse(charArray);
        return new string(charArray);
    }
}

// Usage
string name = "john doe";
bool isEmpty = name.IsNullOrEmpty();
string title = name.ToTitleCase(); // John Doe
string reversed = name.Reverse(); // eod nhoj

Chapter 8: Exception Handling

8.1 Try-Catch-Finally

try
{
    int result = 10 / 0;
}
catch (DivideByZeroException ex)
{
    Console.WriteLine("Division by zero");
    Console.WriteLine($"Error: {ex.Message}");
}
catch (Exception ex)
{
    Console.WriteLine($"General error: {ex.Message}");
}
finally
{
    Console.WriteLine("Cleanup code always executes");
}

8.2 Throwing Exceptions

public class BankAccount
{
    private decimal _balance;
    
    public void Withdraw(decimal amount)
    {
        if (amount <= 0)
        {
            throw new ArgumentException("Amount must be positive");
        }
        
        if (amount > _balance)
        {
            throw new InvalidOperationException("Insufficient funds");
        }
        
        _balance -= amount;
    }
}

8.3 Custom Exceptions

public class InsufficientFundsException : Exception
{
    public decimal Balance { get; }
    public decimal RequestedAmount { get; }
    
    public InsufficientFundsException(
        decimal balance, 
        decimal requestedAmount)
        : base($"Insufficient funds. Balance: {balance}, Requested: {requestedAmount}")
    {
        Balance = balance;
        RequestedAmount = requestedAmount;
    }
}

Chapter 9: Delegates and Events

9.1 Delegates

// Delegate declaration
public delegate int Operation(int a, int b);

// Using delegate
Operation add = (a, b) => a + b;
Operation multiply = (a, b) => a * b;

int result1 = add(5, 3);      // 8

9.2 Events

public class Button
{
    public event EventHandler Click;
    
    public void OnClick()
    {
        Click?.Invoke(this, EventArgs.Empty);
    }
}

public class ButtonHandler
{
    public void HandleClick(object sender, EventArgs e)
    {
        Console.WriteLine("Button clicked!");
    }
}

// Usage
var button = new Button();
var handler = new ButtonHandler();

button.Click += handler.HandleClick;
button.OnClick(); // Button clicked!

Chapter 11: Async/Await

11.1 Async Methods

// Synchronous (blocking)
public string GetData()
{
    Thread.Sleep(1000); // Blocks thread
    return "Data";
}

// Asynchronous (non-blocking)
public async Task GetDataAsync()
{
    await Task.Delay(1000); // Doesn't block thread
    return "Data";
}

11.2 Calling Async Methods

public async Task ProcessDataAsync()
{
    // Sequential execution
    string data1 = await GetDataAsync();
    string data2 = await GetDataAsync();
    
    // Parallel execution
    var task1 = GetDataAsync();
    var task2 = GetDataAsync();
    
    await Task.WhenAll(task1, task2);
    
    Console.WriteLine($"Data1: {task1.Result}");
    Console.WriteLine($"Data2: {task2.Result}");
}

11.3 Cancellation

public async Task ProcessWithCancellationAsync(
    CancellationToken cancellationToken)
{
    try
    {
        await Task.Delay(5000, cancellationToken);
        Console.WriteLine("Operation completed");
    }
    catch (OperationCanceledException)
    {
        Console.WriteLine("Operation was cancelled");
    }
}

// Usage
var cts = new CancellationTokenSource();
var task = ProcessWithCancellationAsync(cts.Token);

// Cancel after 2 seconds
await Task.Delay(2000);
cts.Cancel();

Chapter 12: Memory Management

12.1 Garbage Collection

// GC methods
GC.Collect(); // Force garbage collection (usually not needed)
GC.WaitForPendingFinalizers(); // Wait for finalizers
GC.GetTotalMemory(false); // Get memory usage

// IDisposable pattern
public class ResourceHolder : IDisposable
{
    private IntPtr _unmanagedResource;
    private bool _disposed = false;
    
    public void Dispose()
    {
        Dispose(true);
        GC.SuppressFinalize(this);
    }
    
    protected virtual void Dispose(bool disposing)
    {
        if (!_disposed)
        {
            if (disposing)
            {
                // Dispose managed resources
            }
            // Free unmanaged resources
            CloseHandle(_unmanagedResource);
            _disposed = true;
        }
    }
    
    ~ResourceHolder()
    {
        Dispose(false);
    }
}

12.2 Using Statement

// Traditional way
FileStream file = null;
try
{
    file = new FileStream("test.txt", FileMode.Open);
    // Process file
}
finally
{
    file?.Dispose();
}

// Using statement (recommended)
using (var file = new FileStream("test.txt", FileMode.Open))
{
    // Process file
} // Automatically calls Dispose()

// Multiple using statements (C# 8+)
using var file1 = new FileStream("file1.txt", FileMode.Open);
using var file2 = new FileStream("file2.txt", FileMode.Open);
// Both disposed at end of scope

Chapter 13: Best Practices

13.1 Naming Conventions

// Classes: PascalCase
public class UserService { }

// Methods: PascalCase
public void GetUserById(int id) { }

// Properties: PascalCase
public string UserName { get; set; }

// Fields: camelCase with underscore prefix
private string _userName;

// Local variables: camelCase
string userName = "John";

// Constants: PascalCase
public const int MaxUsers = 100;

// Interfaces: PascalCase with 'I' prefix
public interface IUserService { }

13.2 SOLID Principles

  • Single Responsibility: Each class should have one reason to change
  • Open/Closed: Open for extension, closed for modification
  • Liskov Substitution: Subtypes must be substitutable for base types
  • Interface Segregation: Clients shouldn't depend on unused interfaces
  • Dependency Inversion: Depend on abstractions, not concretions

13.3 Code Organization

  • Keep methods short and focused
  • Use meaningful names for variables and methods
  • Avoid magic numbers - use constants
  • Write unit tests for critical functionality
  • Use XML documentation for public APIs
  • Follow consistent coding style
Tip: Always use async/await for I/O operations to keep your application responsive. Never use async void except for event handlers.
Important: Always dispose of objects that implement IDisposable using the using statement to prevent resource leaks.
