Chapter 1: Introduction to Python

1.1 What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum in 1991, Python emphasizes code readability with its notable use of significant whitespace.

Key Insight: Python is versatile - used in web development, data science, AI, automation, and more.

1.2 Why Python?

Python's popularity stems from its simplicity, extensive libraries, and strong community support.

  • Easy to Learn: Simple syntax similar to English
  • Versatile: Web, data science, AI, automation
  • Large Ecosystem: PyPI with 300,000+ packages
  • Cross-Platform: Runs on Windows, Linux, macOS
  • Strong Community: Active support and resources

1.3 Python Applications

  • Web Development: Django, Flask, FastAPI
  • Data Science: Pandas, NumPy, Matplotlib
  • Machine Learning: TensorFlow, PyTorch, scikit-learn
  • Automation: Scripts, web scraping, testing
  • Desktop Apps: Tkinter, PyQt, Kivy

Chapter 4: Control Flow

4.1 Conditional Statements

Execute different code blocks based on conditions.

# if-elif-else
age = 25
if age < 18:
    print("Minor")
elif age >= 18 and age < 65:
    print("Adult")
else:
    print("Senior")

# Ternary operator
status = "Adult" if age >= 18 else "Minor"

4.2 Loops

Execute code multiple times.

# for loop
for i in range(10):
    print(i)

# while loop
count = 0
while count < 10:
    print(count)
    count += 1

# Loop with else
for i in range(5):
    print(i)
else:
    print("Loop completed")

4.3 Break, Continue, Pass

Control loop execution.

# break
for i in range(10):
    if i == 5:
        break
    print(i)

# continue
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)

# pass (placeholder)
if True:
    pass

Chapter 5: Functions and Modules

5.1 Functions

Reusable blocks of code.

# Function definition
def greet(name):
    return f"Hello, {name}!"

# Function call
message = greet("John")
print(message)

# Default parameters
def greet(name="World"):
    return f"Hello, {name}!"

# Multiple return values
def calculate(a, b):
    return a + b, a - b

sum_result, diff_result = calculate(10, 5)

5.2 Lambda Functions

Anonymous functions for short operations.

# Lambda function
square = lambda x: x ** 2
print(square(5))  # 25

# Lambda with map
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))

# Lambda with filter
even = list(filter(lambda x: x % 2 == 0, numbers))

5.3 Modules and Packages

Organize code into reusable modules.

# Import module
import math
print(math.sqrt(16))  # 4.0

# Import specific function
from math import sqrt
print(sqrt(16))  # 4.0

# Import with alias
import numpy as np
import pandas as pd

5.4 Decorators

Modify function behavior without changing code.

# Simple decorator
def uppercase_decorator(func):
    def wrapper():
        result = func()
        return result.upper()
    return wrapper

@uppercase_decorator
def greet():
    return "hello"

print(greet())  # HELLO

Chapter 6: Object-Oriented Programming

6.1 Classes and Objects

Blueprints for creating objects with properties and methods.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hello, I'm {self.name}"
    
    def __str__(self):
        return f"Person(name={self.name}, age={self.age})"

# Create object
person = Person("John", 25)
print(person.greet())  # Hello, I'm John

6.2 Inheritance

Create new classes based on existing ones.

class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        return f"{self.name} says Woof!"

class Cat(Animal):
    def speak(self):
        return f"{self.name} says Meow!"

6.3 Encapsulation

Hide internal state and require interaction through methods.

class BankAccount:
    def __init__(self, balance):
        self.__balance = balance  # Private attribute
    
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
    
    def get_balance(self):
        return self.__balance

6.4 Polymorphism

Same interface, different implementations.

class Shape:
    def area(self):
        pass

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height
    
    def area(self):
        return self.width * self.height

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius
    
    def area(self):
        return 3.14 * self.radius ** 2

Chapter 7: File Handling and I/O

7.1 Reading Files

Read content from files.

# Read entire file
with open('file.txt', 'r') as file:
    content = file.read()
    print(content)

# Read line by line
with open('file.txt', 'r') as file:
    for line in file:
        print(line.strip())

# Read all lines
with open('file.txt', 'r') as file:
    lines = file.readlines()

7.2 Writing Files

Write content to files.

# Write to file
with open('file.txt', 'w') as file:
    file.write("Hello, World!")

# Append to file
with open('file.txt', 'a') as file:
    file.write("\nNew line")

# Write multiple lines
lines = ["Line 1", "Line 2", "Line 3"]
with open('file.txt', 'w') as file:
    file.writelines('\n'.join(lines))

7.3 Working with Directories

Manage directories and file paths.

import os

# Check if directory exists

Chapter 8: Exception Handling

8.1 Try-Except Blocks

Handle runtime errors gracefully.

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
except Exception as e:
    print(f"An error occurred: {e}")
else:
    print("No errors occurred")
finally:
    print("This always executes")

8.2 Raising Exceptions

Throw exceptions for error conditions.

def divide(a, b):
    if b == 0:
        raise ValueError("Cannot divide by zero")
    return a / b

try:
    result = divide(10, 0)
except ValueError as e:
    print(f"Error: {e}")

8.3 Custom Exceptions

Create domain-specific exceptions.

class InsufficientFundsError(Exception):
    def __init__(self, balance, amount):
        self.balance = balance
        self.amount = amount
        super().__init__(f"Insufficient funds: {balance} < {amount}")

def withdraw(balance, amount):
    if amount > balance:
        raise InsufficientFundsError(balance, amount)
    return balance - amount

Chapter 10: Advanced Python Features

10.1 List Comprehensions

Concise way to create lists.

# Basic comprehension
squares = [x**2 for x in range(10)]

# With condition
evens = [x for x in range(10) if x % 2 == 0]

# Nested comprehension
matrix = [[i*j for j in range(3)] for i in range(3)]

10.2 Generators

Lazy evaluation for memory efficiency.

# Generator function
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# Generator expression
squares = (x**2 for x in range(10))
for square in squares:
    print(square)

10.3 Context Managers

Resource management with with statement.

from contextlib import contextmanager

@contextmanager
def file_manager(filename, mode):
    file = open(filename, mode)
    try:
        yield file
    finally:
        file.close()

with file_manager('file.txt', 'r') as f:
    content = f.read()

10.4 Iterators

Custom iteration over data.

class Countdown:
    def __init__(self, start):
        self.start = start
    
    def __iter__(self):
        return self
    
    def __next__(self):
        if self.start <= 0:
            raise StopIteration
        self.start -= 1
        return self.start + 1

for i in Countdown(5):
    print(i)  # 5, 4, 3, 2, 1

Chapter 13: Best Practices

13.1 Code Style

Follow PEP 8 guidelines for Python code.

  • Indentation: Use 4 spaces per indentation level
  • Naming: snake_case for variables, PascalCase for classes
  • Line length: Maximum 79 characters
  • Imports: Group imports at top of file
  • Whitespace: Use spaces around operators

13.2 Documentation

Document your code with docstrings.

def calculate_area(length, width):
    """
    Calculate the area of a rectangle.
    
    Args:
        length (float): The length of the rectangle
        width (float): The width of the rectangle
    
    Returns:
        float: The area of the rectangle
    """
    return length * width

13.3 Error Handling

Handle exceptions gracefully.

  • Use specific exceptions instead of broad Exception
  • Log errors for debugging
  • Provide meaningful error messages
  • Use finally for cleanup
Tip: Use type hints for better code documentation and IDE support.
Important: Always use virtual environments to manage project dependencies.
Final Advice: Write clean, readable code. Python emphasizes readability - make your code self-documenting.
